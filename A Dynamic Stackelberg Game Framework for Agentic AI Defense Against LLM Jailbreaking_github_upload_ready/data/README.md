# Data Notes

- `heldout_benign_prompts.json` contains the benign prompts used for the public benign-side checks.
- `public_prompt_space.json` is the redacted public prompt library used by default in this release.
- `restricted_prompt_space.template.json` is a placeholder template for users who have approved access to a non-public prompt library and need to override `PROMPT_SPACE_PATH`.
- Raw harmful generations and raw trajectory dumps from the private experiments were removed from the public release.
